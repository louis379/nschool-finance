# Changelog

## [Unreleased]

### 2026-03-22 — 記帳頁 UX 優化

- **點擊刪除**：點選交易列即可滑出刪除按鈕，刪除後顯示「復原」按鈕（3.5 秒內可撤回）
- **記住上次分類**：開啟新增彈窗時自動帶入上次使用的分類，減少重複點選
- **日期選擇器**：新增記錄時可選擇日期（預設今天），支援補登過去的支出
- autoFocus 金額欄位，開啟彈窗即可直接輸入

## [2026-03-22] — 模擬交易 (Stock Trading)
- **快速選擇數量**：交易彈窗新增一鍵預設數量按鈕；買入可選 ×1 / ×5 / ×10 / 最多，賣出可選 ¼ / ½ / 全部，省去手動輸入步驟
- **市場情緒條**：股票列表上方加入上漲/下跌家數指示條（進度條顯示漲跌比例），一眼掌握當前市場情緒
- **切換 Tab 淡入動畫**：切換台股/美股/加密貨幣時，股票列表以 fadeInUp 動畫重新載入，強化即時感
- Reference: Robinhood（快速預設）、moomoo（情緒條）、Webull（Tab 動畫）
