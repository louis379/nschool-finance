居住：台北市

「一個月後，我能自信地說出為什麼我要買 0050 而不是 2330」

## 不是我們的用戶

以下用戶類型**不是**我們的目標，不要為他們設計：

- ❌ 資深投資者（自營交易、當沖客）→ 他們要 Bloomberg/XQ
- ❌ 完全不在乎投資的人 → 說服成本太高
- ❌ 50 歲以上投資者 → 不同的資訊消費習慣
- ❌ 投機者（想快速致富）→ 我們強調教育，不是明牌

|------|------|

| 安全感需求 | 先在安全環境練習，有信心再實際操作 |
| 社群影響 | 朋友推薦和口碑很重要 |

## 用戶旅程地圖

```
認知 → 安裝 → 首次體驗 → 習慣養成 → 付費 → 推薦

[認知]
觸點：朋友推薦、App Store 搜尋「股票學習」、
      Instagram 廣告、財經 YouTuber 提及

[安裝]
目標：第一次打開 App ≤ 1 分鐘完成帳號建立

[首次體驗（D0-D3）]
關鍵：完成投資程度問卷 → AI 推薦第一課 → 完成一堂課
      → 做第一筆模擬交易
危機：放棄原因通常是「不知道該做什麼」

[習慣養成（D4-D30）]
關鍵：每日提醒 → 學習連續天數徽章 → 模擬帳戶有損益
      → 情感連結建立
危機：學習內容太難或太無聊

[付費轉換（D30+）]
觸發：免費功能限制 → 「付費解鎖進階課程」
需確認：定價策略（待人決策）

[推薦]
觸發：在社群分享持倉截圖 → 告訴朋友「我在用這個學投資」
```
