# 競品分析

> nSchool Finance 的市場定位與競品比較

---

## 市場定位

nSchool Finance 定位在「投資理財學習」和「個人記帳」的交集處。不同於純記帳 App 或純投資平台，我們的核心是「透過學習和模擬來建立投資能力」。

```
        純記帳            nSchool Finance            純投資
    ┌───────────┐     ┌────────────────┐      ┌───────────┐
    │ 記帳雞    │     │ 記帳 + 學習    │      │ 國泰證券  │
    │ CWMoney  │ ──→ │ + 模擬交易    │ ──→  │ 富邦證券  │
    │ MoneyBook│     │ + AI 分析      │      │ 永豐金    │
    └───────────┘     └────────────────┘      └───────────┘
```

## 我們的差異化

1. **記帳 + 學習一體化** — 不需要裝兩個 App
2. **AI 個人化分析** — 根據用戶的實際財務狀況給建議
3. **零風險模擬** — 用虛擬資金練習，不怕虧
4. **專為新手設計** — UI 極度簡化，用白話文教學
5. **AI 驅動的持續改善** — 產品迭代速度遠超傳統團隊

## 市場動態追蹤

### 2026-03-25 更新

**CWMoney（超過 500 萬用戶）：**
- 最新版本 5.1.1（Android 於 2026-01-09 更新）
- 更新內容主要為技術維護：升級 TargetSDK 35、修復 Android 14+ 快速記帳 Widget 問題
- 目前**尚未加入 AI 功能**，仍以傳統記帳為核心，這是我們的差異化窗口
- 觀察：CWMoney 更新節奏放慢，以維護為主，非積極擴展新功能

**MOZE 4.0：**
- 2026 年初調整訂閱價格回歸原價：Premium $1.99/月、Pro Max $3.99/月
- 新增 Apple Watch 記帳支援（含名稱、商家、標籤、獎勵、備註）
- 新增紀錄匯出為圖片功能、帳戶統計報表摘要頁籤
- 支援加密貨幣追蹤、iPad 版本最佳化、跨裝置同步
- 觀察：MOZE 在記帳體驗打磨上持續投入，但仍不碰投資學習領域

**CMoney 理財寶：**
- 持續經營投資工具生態（籌碼K線、選股工具等）
- 目標用戶偏中高級投資者，與我們的新手定位不衝突
- 未發現 2026 年重大新功能發布

**eToro 台灣：**
- 在台灣仍未註冊及受監管，屬於境外平台
- 最低入金 50 美元，銀行轉帳最低 500 美元
- 新推出 24/7 石油交易功能
- 風險：台灣用戶若發生糾紛無法透過本地法律維權
- 觀察：監管問題持續是 eToro 在台灣的阻力，對我們不構成直接威脅

**2026 台灣記帳 APP 市場趨勢：**
- 「半自動記帳」成為主流：大額用同步、小額手動記
- AI 功能趨勢：本地 AI 處理（避免上傳財務數據到不明伺服器）、AES-256 加密標準
- 「預測性理財報表」成新賣點：根據 3 個月消費習慣預測月中超支風險
- 碎碎念記帳（對話式記帳）以個性化互動體驗突圍
- 麻布記帳（Moneybook）以開放銀行自動同步維持領先
- **關鍵洞察：市場上尚未出現「記帳 + 投資學習 + AI 分析」三合一的產品，我們的定位仍然獨特**

## 威脅與機會

### 威脅
- 大型記帳 App 加入投資功能（目前尚未發生，但需持續監控）
- 券商 App 加入教學功能
- AI 競品快速出現（碎碎念記帳等新玩家以 AI 互動切入）
- 「預測性理財」功能可能被競品快速複製
- 隱私安全意識提升，用戶對 AI 分析財務數據更謹慎

### 機會
- 台灣年輕人理財意識持續提升
- AI 技術讓小團隊能做出大產品
- 「先學再投」的需求尚未被好好滿足
- CWMoney 更新放緩，存在用戶轉移機會
- 本地 AI 處理趨勢與我們的技術方向一致
- 「預測性理財報表」可作為差異化功能快速實現
- MOZE 定價回升，免費用戶可能尋找替代方案
