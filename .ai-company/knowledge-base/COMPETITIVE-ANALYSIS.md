# 競品分析

> nSchool Finance 的市場定位與競品比較

---

## 市場定位

nSchool Finance 定位在「投資理財學習」和「個人記帳」的交集處。不同於純記帳 App 或純投資平台，我們的核心是「透過學習和模擬來建立投資能力」。

```
        純記帳            nSchool Finance            純投資
    ┌───────────┐     ┌────────────────┐      ┌───────────┐
    │ 記帳雞    │     │ 記帳 + 學習    │      │ 國泰證券  │
    │ CWMoney  │ ──→ │ + 模擬交易    │ ──→  │ 富邦證券  │
    │ MoneyBook│     │ + AI 分析      │      │ 永豐金    │
    └───────────┘     └────────────────┘      └───────────┘
```

## 我們的差異化

1. **記帳 + 學習一體化** — 不需要裝兩個 App
2. **AI 個人化分析** — 根據用戶的實際財務狀況給建議
3. **零風險模擬** — 用虛擬資金練習，不怕虧
4. **專為新手設計** — UI 極度簡化，用白話文教學
5. **AI 驅動的持續改善** — 產品迭代速度遠超傳統團隊

## 威脅與機會

### 威脅
- 大型記帳 App 加入投資功能
- 券商 App 加入教學功能
- AI 競品快速出現

### 機會
- 台灣年輕人理財意識提升
- AI 技術讓小團隊能做出大產品
- 「先學再投」的需求尚未被好好滿足
