## 格式

```markdown
### [BUG-XXX] 標題

- **嚴重度**：P0（阻斷）/ P1（嚴重）/ P2（一般）/ P3（輕微）
- **影響**：[哪些用戶/功能受影響]
- **重現步驟**：[如何重現]
- **預期行為**：[應該是怎樣]
- **實際行為**：[現在是怎樣]
- **狀態**：待修復 / 修復中 / 已修復 / 無法重現
- **修復日期**：[日期]
```

## 目前已知的 Bug

### BUG-001 手機端記帳金額輸入鍵盤遮擋確認按鈕

- **嚴重度**：P1
- **影響**：iOS Safari 用戶在記帳時無法看到確認按鈕
- **重現步驟**：手機開啟記帳頁 → 點擊金額欄位 → 虛擬鍵盤彈出 → 確認按鈕被遮住
- **預期行為**：頁面應自動滾動，讓確認按鈕可見
- **實際行為**：確認按鈕被鍵盤完全遮住
- **狀態**：待修復

### BUG-002 儀表板數據載入時閃爍

- **嚴重度**：P2
- **影響**：所有用戶首次載入儀表板時會看到數字從 0 跳到實際值
- **重現步驟**：登入後進入儀表板 → 觀察數字顯示
- **預期行為**：應顯示 skeleton 載入狀態，數據就緒後平滑顯示
- **實際行為**：先顯示 0，然後突然跳到實際數值
- **狀態**：待修復
